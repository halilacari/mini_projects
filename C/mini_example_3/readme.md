Buradaki kodlar girilen satır sayısına göre : 

örnek: girilen sayı 5 olsun.

çıktı: a b c d 5
çıktı: b c d 5 4
çıktı: c d 5 4 3
çıktı: d 5 4 3 2
çıktı: 5 4 3 2 1


örnek : girilen sayı 4 olsun.

çıktı: a b c 4
çıktı: b c 4 3
çıktı: c 4 3 2
çıktı: 4 3 2 1


örnek : girilen sayı 3 olsun.

çıktı: a b 3
çıktı: b 3 2
çıktı: 3 2 1


şeklinde çıktı vermesi üzerine yazılmıştır. C dili kullanılmıştır.
